#  Bike Store Sales Analysis

# Project Overview
This project analyzes Bike Store sales data to extract meaningful business insights using **SQL, Excel, and Tableau**.  
The focus is on building a clean analytical dataset through **SQL joins and aggregations**, followed by dashboard creation for decision-making support.

The analysis covers:
- Total revenue
- Monthly sales trends
- Revenue by state
- Revenue by brand and product category
- Store-level performance
- Top 10 products by revenue
