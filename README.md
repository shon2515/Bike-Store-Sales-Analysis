# Bike-Store-Sales-Analysis
This project analyzes Bike Store sales data using SQL, Excel, and Tableau to extract key business insights. It covers total revenue, monthly trends, sales by state, brand, category, store performance, and top products. SQL was used for data cleaning, joins, aggregations, Excel for pivot dashboards, and Tableau for interactive visualizations.


### Data Preparation
- Raw data stored in SQL Server
- Final analytical dataset created using SQL joins and aggregations
- Exported SQL query results to CSV format
- CSV file used for Excel and Tableau dashboards
